# Learing Tailwind CSS with Building a Simple LAnding Page

## Steps 

- Inititialised npm
- Installed tailwind
- Created CSS Folder in root 
- Added tailwind directives in the style.css

Installation

    npm init -y
    npm install tailwindcss

Add Tailwind directives in root css file 

    @tailwind base;
    @tailwind components;
    @tailwind utilities;
